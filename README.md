## :dolphin:介绍
### 该项目为Python实时爬取[B站直播](https://live.bilibili.com/)弹幕并发送跟随弹幕
- 项目介绍:通过传入用户登录B站后得到的Cookie(可以登录后抓包获取)，即可实现监控B站直播弹幕并发送跟随弹幕
- 运行方式:运行Spiders目录下bilibili_live_barrage.py文件即可
## :dolphin:运行环境
Version: Python3
## :dolphin:安装依赖库
```
pip3 install -r requirements.txt
```
